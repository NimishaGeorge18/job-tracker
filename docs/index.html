Job Application Tracker (CLI)

Python command-line application to track job applications with persistent JSON storage.

Features

  • Add / View / Update / Delete applications
  • Filter by status
  • Stats + interview rate
  • JSON persistence
  • Search by company

Run Locally

python3 main.py
