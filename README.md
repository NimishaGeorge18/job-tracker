# Job Application Tracker (CLI)

A simple CLI tool to track job applications. Add applications, update status, filter, delete, and view summary stats. Data is saved in a JSON file.

## Features

- Add new application
- View all applications
- Update status (Applied / Interview / Rejected / Offer)
- Filter by status
- Delete an application
- Stats summary + interview rate
- Persistent storage in JSON

## Tech Stack

- Python
- JSON file storage

## How to Run

```bash
python3 main.py
```
